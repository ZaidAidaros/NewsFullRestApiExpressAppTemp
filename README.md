# News Full Rest API <br/>
### This is News full rest api template for developping news backend apps using express <br/>
## To Install:
After clone it
```
#change dir
cd NewFullRestApiExpressAppTemp
# yarn
yarn

# npm
npm install

# pnpm
pnpm install
```
## To Preview
```
# yarn
yarn dev

# npm
npm run dev

# pnpm
pnpm dev
```
